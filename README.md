## Project Description

This repository contains solutions to the COS 202 Python programming assignment.

### Project 1: Mathematical Calculator (MC)
A GUI-based calculator developed using Python Tkinter. It performs the following operations:
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Modulus (%)
- Exponentiation (^)
- Clear (C)
- OFF

### Project 2: Personal Pocket CGPA Calculator (PPC)
A Python program that calculates a student's CGPA by accepting course units and grades. It uses `if`, `elif`, and `else` selection control statements to calculate the CGPA and display the corresponding class of degree.
